JavaScript Quick Reference

Intermediate Level

Variables

let height = 5.4;
const FPS = 30;  // Can't change

const is recommended for all values that won't change.

String concatenation

"5" + "5"
⇒
"55"
"5" - "5"
⇒
0

Use parseInt, parseFloat, or unary plus to prevent concatenation. 

const userInput = '19';
const userAge = +userInput;      //19;
const ageNextYear = userAge + 1; //20;

Template strings include variables concisely

const text = `In one year,
  you will be ${ageNextYear}`;

Loops

for (let i = 0; i < 10; i++) {
  console.log(i);
}
while (current = queue.getNext()) {
  console.log(current.value);
}
do {
  input = prompt('Enter a number');
} while(isNaN(input));

Array functions (not loops) are recommended for traversing arrays.

2 + 2

4

⇒
3

⇒
20

Use Math.floor for integer arithmetic

const eggs = 29;
const cartons = Math.floor(eggs / 12);
const leftover = eggs % 12;

Functions

function declaration

function showResult(text) {
  document.body.innerText = text;
}

arrow function

const showResult = (text) => {
  document.body.innerText = text;
}

function declaration

function getCirc(r) {
  return 2 * Math.PI * r;
}

arrow function

const getCirc = (r) => 2*Math.PI*r;

Arrow functions are concise inside callbacks.

setTimeout(() => {
  console.log('1s later');
}, 1000);

Traversing Arrays

const myList = [1, 2, 5, 7, 4];
myList.filter(v => v > 2)
⇒
[5, 7, 4]
myList.map(v => v * 2)
⇒
[2, 4, 10, 14, 8]
myList.reduce((a, b) => a + b)
⇒
19
myList.forEach(v => console.log(v))
⇒
(Prints each value in the list)

Additional Array Functions

['cooking', 'hiking', 'skiing'].includes('cooking')
⇒
true
